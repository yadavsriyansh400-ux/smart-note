# 📝 Smart Notes App

A full-stack notes application built using **React, Node.js, Express, and MongoDB** with a modern blue UI.

---

## 🚀 Features

* Create notes
* View notes
* Update notes (inline editing modal)
* Delete notes
* Persistent storage using MongoDB
* Smooth UI with hover effects and modal (island view)

---

## 🛠 Tech Stack

* Frontend: React (Vite)
* Backend: Node.js + Express
* Database: MongoDB (Mongoose)

---

## 📦 Installation

### 1. Clone repo

git clone https://github.com/your-username/smart-notes-app.git

### 2. Backend setup

cd server
npm install

Create `.env` file:
MONGO_URI=your_mongodb_connection_string

Run server:
npm run dev

---

### 3. Frontend setup

cd client
npm install
npm run dev

---

## 🌐 Usage

Open:
http://localhost:5173
